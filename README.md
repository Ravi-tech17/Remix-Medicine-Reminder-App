# 📖 Overview

**MedRemind** is a smart medicine reminder application designed to help users take medications on time.

The application allows users to schedule medicines, receive reminders, track medication history, monitor daily progress, and manage prescriptions through a clean and user-friendly interface.

Whether you're managing your own medications or helping a family member, MedRemind makes medication management simple and reliable.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/MedRemind.git
```

## Navigate to Project

```bash
cd MedRemind
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file.

```env
VITE_FIREBASE_API_KEY=your_api_key

VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain

VITE_FIREBASE_PROJECT_ID=your_project_id

VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket

VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id

VITE_FIREBASE_APP_ID=your_app_id
```

## Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
